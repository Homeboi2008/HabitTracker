# Трекер Привычек (Habit Tracker)

## Краткое описание

Мобильное приложение для отслеживания ежедневных привычек. Пользователь может добавлять привычки, отмечать их выполнение и просматривать статистику. Приложение помогает формировать полезные привычки через визуальное отслеживание прогресса.

---

## Основные функции

### Базовые функции (MVP)
1. **Добавление привычек** — создание новой привычки с названием и описанием
2. **Список привычек** — отображение всех добавленных привычек
3. **Отметка выполнения** — возможность отметить выполнение привычки за сегодня
4. **Удаление привычек** — удаление ненужных привычек из списка
5. **Статистика** — просмотр общего количества выполненных дней для каждой привычки
6. **Мотивационные цитаты** — загрузка вдохновляющих цитат через API

## Структура экранов

### 1. Главный экран
**Назначение:** Отображение списка всех привычек пользователя

**Элементы интерфейса:**
- Заголовок "Мои Привычки"
- Кнопка "Добавить привычку" (вверху справа или внизу)
- Список привычек (каждая карточка содержит):
  - Название привычки
  - Описание (опционально)
  - Индикатор выполнения на сегодня (чекбокс или кнопка)
  - Счетчик выполненных дней (например, "7/30 дней")

## Навигация

### Схема навигации:
```
                    [Главный экран]
                         │
        ┌────────────────┼────────────────┐
        │                │                │
   [Добавить]    [Статистика]    [Карточка привычки]
   (экран)       (экран)         (детали/редактирование)
        │                │
        └────────────────┘
              (назад)
```

### Переходы:
1. **Главный экран → Экран добавления:**
   - По нажатию кнопки "Добавить привычку"
   - Возврат по кнопке "Отмена" или стрелке назад

2. **Главный экран → Экран статистики:**
   - По нажатию кнопки/иконки "Статистика"
   - Возврат по стрелке назад

3. **Главный экран → Редактирование привычки:**
   - По долгому нажатию на карточку привычки (опционально)
   - Возврат по стрелке назад

## Сценарии использования

### Сценарий 1: Добавление новой привычки
1. Пользователь открывает приложение (главный экран)
2. Нажимает кнопку "Добавить привычку"
3. Вводит название привычки (например, "Читать 30 минут")
4. Вводит описание (опционально)
5. Нажимает "Сохранить"
6. Возвращается на главный экран, видит новую привычку в списке

### Сценарий 2: Отметка выполнения привычки
1. Пользователь на главном экране видит список привычек
2. Нажимает на чекбокс/кнопку выполнения у нужной привычки
3. Чекбокс меняется на отмеченный (☑)
4. Счетчик выполненных дней увеличивается на 1
5. При повторном нажатии можно снять отметку (уменьшить счетчик)

### Сценарий 3: Просмотр статистики
1. Пользователь на главном экране нажимает "Статистика"
2. Видит список всех привычек с процентами выполнения
3. Видит визуальные индикаторы прогресса
4. Видит мотивационную цитату (загруженную через API)
5. Может обновить цитату кнопкой "Обновить цитату"

### Сценарий 4: Удаление привычки
1. Пользователь на главном экране видит список привычек
2. Нажимает на иконку корзины у нужной привычки
3. Появляется диалог подтверждения "Удалить привычку?"
4. Подтверждает удаление
5. Привычка исчезает из списка

### Сценарий 5: Загрузка мотивационной цитаты
1. Пользователь открывает экран статистики
2. Приложение автоматически загружает цитату через API при открытии экрана
3. Пользователь видит цитату
4. При нажатии "Обновить цитату" загружается новая цитата

## Технические детали (для разработки)

### Структура данных привычки:
```dart
class Habit {
  String id;           // Уникальный идентификатор
  String title;        // Название привычки
  String description;  // Описание (опционально)
  int completedDays;   // Количество выполненных дней
  int targetDays;      // Целевое количество дней (по умолчанию 30)
  Map<DateTime, bool> dailyProgress; // История выполнения по датам
}
```

### API для мотивационных цитат:
- **Quotable API:** `https://api.quotable.io/random`
- **Формат ответа:** JSON
- **Пример ответа:**
```json
{
  "_id": "string",
  "content": "Цитата",
  "author": "Автор"
}
```

### Цветовая схема:
- **Основной цвет:** #2196F3 (синий)
- **Акцентный цвет:** #4CAF50 (зеленый для выполненных)
- **Фон:** #FFFFFF (белый) / #F5F5F5 (светло-серый)
- **Текст:** #212121 (темно-серый)
- **Вторичный текст:** #757575 (серый)

Приложение считается готовым, если:
1.  Можно добавить новую привычку
2.  Можно просмотреть список всех привычек
3.  Можно отметить выполнение привычки
4.  Можно удалить привычку
5.  Можно просмотреть статистику
6.  Можно загрузить мотивационную цитату через API
7.  Данные сохраняются между запусками приложения
8.  Все экраны имеют базовую стилизацию
